> AUTH_FLOW

  1. POST /v1/auth/login -- send credentials, receive HttpOnly cookies.
  2. GET /v1/auth/me -- read access_token cookie, return profile.
  3. POST /v1/auth/refresh -- read refresh_token cookie, renew access_token.
  4. POST /v1/auth/logout -- clear both cookies.
